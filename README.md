# Backend Stockarium

Backend de la plataforma Stockarium.

Este repositorio contiene la lógica de negocio, gestión de base de datos, autenticación y servicios backend que soportan el sistema de administración de stock e inventario.

---

## Responsabilidades

- Gestión de base de datos  
- Lógica de negocio  
- Autenticación y autorización  
- Persistencia de datos  
- Exposición de endpoints / servicios para frontend  

---

## Tecnologías Utilizadas
- Java Spring Boot
- PostgreSQL / Supabase 
- JWT / Auth Provider
- Docker

---

## Funcionalidades

- CRUD de productos / inventario  
- Gestión de usuarios  
- Validaciones de negocio  
- Integración con frontend de Stockarium  

---

## Objetivo del Proyecto

Proveer una capa backend robusta y escalable para centralizar la gestión de stock y operaciones de inventario.
